    1. Import WaveFile:
       ```typescript

       ```

    2. Define AudioFixtureOptions interface:
       ```typescript
       export interface AudioFixtureOptions {
         duration?: number;      // seconds, default: 1
         sampleRate?: number;    // Hz, default: 44100
         channels?: 1 | 2;       // mono or stereo, default: 2
         bitDepth?: 8 | 16 | 24 | 32; // default: 16
         frequency?: number;     // Hz for sine wave tone, default: 440 (A4 note)
         volume?: number;        // 0-1, default: 0.5
       }
       ```

    3. Create AudioFactory class with static methods:

       a) `createWAV(options?: AudioFixtureOptions): Buffer`
          - Generate sine wave samples:
            ```typescript
            const numSamples = Math.floor((options.duration ?? 1) * (options.sampleRate ?? 44100));
            const channels = options.channels ?? 2;
            const samples = new Int16Array(numSamples * channels);
            const frequency = options.frequency ?? 440;
            const volume = options.volume ?? 0.5;
            const sampleRate = options.sampleRate ?? 44100;

            for (let i = 0; i < numSamples; i++) {
              const t = i / sampleRate;
              const value = Math.sin(2 * Math.PI * frequency * t) * 32767 * volume;
              for (let ch = 0; ch < channels; ch++) {
                samples[i * channels + ch] = value;
              }
            }

            const wav = new WaveFile();
            wav.fromScratch(channels, sampleRate, '16', samples);
            return Buffer.from(wav.toBuffer());
            ```

       b) `createSilentWAV(options?: Omit<AudioFixtureOptions, 'frequency' | 'volume'>): Buffer`
          - Create silent audio (useful for testing audio without actual sound)
          - Same as createWAV but with all zero samples

       c) `createVariations(): Record<string, Buffer>`
          - Return object with edge case audio files:
            - silent: 1 second of silence
            - mono: 1 second mono (1 channel)
            - stereo: 1 second stereo (2 channels)
            - short: 0.1 second (100ms)
            - long: 5 seconds
            - lowSampleRate: 22050 Hz
            - highSampleRate: 48000 Hz
            - lowFreq: 100 Hz tone
            - highFreq: 8000 Hz tone

       d) `getSampleCount(buffer: Buffer): number`
          - Parse WAV and return sample count (for validation in tests)
          - Use WaveFile to parse and read

       e) `getDuration(buffer: Buffer): number`
          - Parse WAV and return duration in seconds
          - Calculate from sample count and sample rate

    Note: Only WAV generation is needed - the app converts WAV to other formats (FLAC, MP3, OGG, Opus).
    We don't need to generate those formats for testing.
