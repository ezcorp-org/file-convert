**Factory methods:**
- `createZeroByteFile(filename: string): Buffer` - Empty file (0 bytes)
- `createTruncatedFile(format: string, bytes: number): Buffer` - Valid header but truncated body
- `createBadHeaderFile(claimedFormat: string): Buffer` - Random bytes claiming to be a format
- `createOversizedMetadata(format: string, sizeMB: number): { buffer: Buffer, metadata: { name: string, mimeType: string } }` - Returns buffer with file metadata for size limit testing
- `createSpoofedExtension(actualFormat: string, claimedExtension: string): { buffer: Buffer, filename: string, mimeType: string }` - Valid file with wrong extension

**Implementation notes:**
- Use MAGIC_SIGNATURES from magic-bytes.ts for valid header bytes
- For truncated files: write valid header + 10-50 random bytes (not full valid file)
- For bad header: write random bytes that don't match any known signature
- For spoofed extension: use ImageFactory/AudioFactory to create real file, return with wrong extension
- Export CorruptedFileFactory from factories/index.ts
