Use `page.evaluateHandle()` to create DataTransfer in browser context as documented in research:

```typescript
test('uploads PNG via drag-and-drop', async ({ page }) => {
  const pngBuffer = await ImageFactory.createPNG({ width: 100, height: 100 });

  await page.goto('/convert');
  await page.waitForLoadState('networkidle');

  // Create DataTransfer in browser context
  const dataTransfer = await page.evaluateHandle((bufferArray) => {
    const dt = new DataTransfer();
    const file = new File([new Uint8Array(bufferArray)], 'dropped.png', { type: 'image/png' });
    dt.items.add(file);
    return dt;
  }, [...pngBuffer]);

  // Find drop zone and dispatch drop event
  const dropZone = page.locator('.drop-zone');
  await dropZone.dispatchEvent('drop', { dataTransfer });

  // Verify file appeared
  await expect(page.locator('.file-item')).toContainText('dropped.png');
  await expect(page.locator('.configure-section')).toBeVisible();
});
```

Test cases to include:
1. Single PNG file via drag-and-drop
2. Single JPEG file via drag-and-drop
3. Multiple files via drag-and-drop (2-3 PNGs)

Note: Drag-and-drop tests are Chromium-only per research recommendations. Add test.skip for Firefox/WebKit if browser-specific issues occur.

The drop zone selector may need adjustment based on actual UI. Check FileUploader.svelte for the correct class/element.
