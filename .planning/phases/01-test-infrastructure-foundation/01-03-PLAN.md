  <name>Task 1: Create GitHub Actions workflow for E2E tests</name>
  <files>.github/workflows/e2e-tests.yml</files>
  <action>
Create .github/workflows/ directory if needed, then create e2e-tests.yml:

```yaml
name: E2E Tests

on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

jobs:
  e2e-tests:
    runs-on: ubuntu-latest
    timeout-minutes: 30

    steps:
      - uses: actions/checkout@v4

      - name: Setup Bun
        uses: oven-sh/setup-bun@v2
        with:
          bun-version: latest

      - name: Install dependencies
        run: |
          bun install
          cd apps/frontend && bun install

      - name: Get Playwright version
        id: playwright-version
        run: |
          cd apps/frontend
          PLAYWRIGHT_VERSION=$(bun pm ls @playwright/test 2>/dev/null | grep @playwright/test | sed 's/.*@//')
          echo "version=$PLAYWRIGHT_VERSION" >> $GITHUB_OUTPUT

      - name: Cache Playwright browsers
        uses: actions/cache@v4
        id: playwright-cache
        with:
          path: ~/.cache/ms-playwright
          key: playwright-${{ runner.os }}-${{ steps.playwright-version.outputs.version }}

      - name: Install Playwright browsers
        if: steps.playwright-cache.outputs.cache-hit != 'true'
        run: cd apps/frontend && npx playwright install --with-deps chromium

      - name: Install Playwright system dependencies
        if: steps.playwright-cache.outputs.cache-hit == 'true'
        run: cd apps/frontend && npx playwright install-deps chromium

      - name: Run E2E tests
        run: cd apps/frontend && bun run test:e2e
        env:
          CI: true

      - name: Upload test results
        uses: actions/upload-artifact@v4
        if: failure()
        with:
          name: playwright-report
          path: apps/frontend/playwright-report/
          retention-days: 7

      - name: Upload test traces
        uses: actions/upload-artifact@v4
        if: failure()
        with:
          name: test-traces
          path: apps/frontend/test-results/
          retention-days: 7
```

Key decisions:
- Uses Bun (matches project setup - see package.json scripts)
- Caches Playwright browsers by version to speed up subsequent runs
- Only installs Chromium (primary browser) to reduce CI time
- Uploads reports and traces on failure for debugging
- Sets CI=true environment variable for Playwright config detection
