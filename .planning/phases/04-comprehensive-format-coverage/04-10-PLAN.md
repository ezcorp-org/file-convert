2. Verify it has a handler for TXT output format
3. If missing, add simple text extraction:

For HTML to TXT:
```javascript
case 'txt':
  if (job.fromFormat === 'html') {
    // Strip HTML tags, decode entities, preserve line breaks
    let text = content
      .replace(/<br\s*\/?>/gi, '\n')
      .replace(/<\/p>/gi, '\n\n')
      .replace(/<\/div>/gi, '\n')
      .replace(/<\/h[1-6]>/gi, '\n\n')
      .replace(/<li>/gi, '- ')
      .replace(/<\/li>/gi, '\n')
      .replace(/<[^>]+>/g, '')  // Remove remaining tags
      .replace(/&nbsp;/g, ' ')
      .replace(/&amp;/g, '&')
      .replace(/&lt;/g, '<')
      .replace(/&gt;/g, '>')
      .replace(/&quot;/g, '"')
      .replace(/\n{3,}/g, '\n\n')  // Normalize multiple newlines
      .trim();
    outputBlob = new Blob([text], { type: 'text/plain' });
  }
  mimeType = 'text/plain';
  break;
```

For MD to TXT:
```javascript
// Markdown is already mostly plain text, just remove markdown syntax
if (job.fromFormat === 'md' || job.fromFormat === 'markdown') {
  let text = content
    .replace(/^#+\s*/gm, '')  // Remove heading markers
    .replace(/\*\*([^*]+)\*\*/g, '$1')  // Remove bold
    .replace(/\*([^*]+)\*/g, '$1')  // Remove italic
    .replace(/\[([^\]]+)\]\([^)]+\)/g, '$1')  // Remove links, keep text
    .replace(/`([^`]+)`/g, '$1')  // Remove code markers
    .replace(/^[-*]\s+/gm, '- ')  // Normalize list markers
    .trim();
  outputBlob = new Blob([text], { type: 'text/plain' });
}
```

AVOID: Complex parsing libraries that add significant bundle size.
WHY: Text extraction is simple enough for regex-based approach.
